# Healthcare Dashboard UI

## 🚀 About Project
This is a responsive Healthcare Dashboard UI built using HTML, CSS, and JavaScript.  
The dashboard allows users to view patient details, diagnosis history, and health metrics in a clean and organized interface.

## 🛠️ Tech Stack
- HTML
- CSS
- JavaScript
- Chart.js

## ✨ Features
- Patient list sidebar
- Dynamic patient profile update on selection
- Diagnosis history section
- Health metrics cards (Respiratory Rate, Temperature, Heart Rate)
- Lab results section
- Blood pressure chart visualization using Chart.js
- Clean and modern healthcare dashboard UI

## 📦 How to Run Locally
1. Download or clone the repository
2. Open `index.html` in your browser
